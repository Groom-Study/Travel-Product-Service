# ☁️ Groom Deep Dive Fullstack 19th - Team 3: NOT NULL

> **"비어있지 않은 열정, 반드시 결과를 만드는 NOT NULL"** > **📋 팀 미션: AWS Serverless 기반 여행 상품 커머스 서비스 구현**

React와 Context API를 활용하여 여행 상품을 판매하는 커머스 웹 서비스를 구현하였습니다. 내부 REST API를 직접 설계/구현하고 자동화 테스트 및 AWS 배포까지 경험합니다.

### 🎯 핵심 습득 역량
