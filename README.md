# 🚀 여행 상품 만들기 - 구름 딥다이브 풀스택 19회차 3팀 **NOT_NULL** EXP 미션

## 📋 프로젝트 개요

- **미션 명**: 여행 상품 만들기 (EXP 미션)
- **팀 이름**: 3팀 NOT_NULL
- **주요 기능**:
  - 여행 상품 관리 및 주문 시스템
  - 실시간 가격 계산 및 상품 수량 조절
  - 미디어 쿼리를 활용한 응답형 그리드 레이아웃 (1단 ~ 5단 구성)

## 🛠 기술 스택

- **Frontend**: React, TypeScript, Vite
- **Styling**: Vanilla CSS (CSS Modules/Global)
- **HTTP Client**: Axios

## ⚙️ 실행 방법

### 1. 개발 서버 실행

```bash
npm install
npm run dev
```

### 2. 백엔드 서버 (Mock API) 실행

_백엔드 서버가 로컬환경(`http://localhost:3300`)에서 실행 중이어야 합니다._

## 📂 폴더 구조

- `src/components`: 공용 컴포넌트
- `src/pages`: 페이지별 컴포넌트 (`OrderPage`, `SummaryPage`, `CompletePage`)
- `src/contexts`: 컨텍스트 API (주문 데이터 관리)
- `src/index.css`: 스타일 CSS
